# 表ADT

一个表有N个相同类型的元素. 称这个表的大小为N. 大小为0的表为空表.

表中的第一个元素是$A_1$,最后一个元素是$A_N$

1. 前驱和后继

    $A_{i+1}$是$A_i$的后继元,$A_{i-1}$是$A_i$的前驱元($1 < i < N$),
    
    通常来说,我们不定义$A_1$的前驱元,也不定义$A_N$的后继元

1. 基本操作
    * PrintList
    * MakeEmpty
    * Find
    * Insert
    * Delete
    * FindKth
